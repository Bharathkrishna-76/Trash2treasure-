## 🚀 Deploy to GitHub Pages in 4 Steps

1. **Create repo:** `github.com/new` → name `t2t` → Public → Create
2. **Upload files:** Add file → Upload → drag `index.html` + `README.md` → Commit
3. **Enable Pages:** Settings → Pages → main branch → root → Save
4. **Visit:** `https://YOUR-USERNAME.github.io/t2t/`
