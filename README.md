## 🚀 Deploy in 4 Steps

### 1. Create GitHub Repository
- Go to [github.com/new](https://github.com/new)
- Name: `t2t` · Visibility: **Public** · Click **Create repository**

### 2. Upload Files
- Click **Add file → Upload files**
- Drag `index.html` and `README.md` → **Commit changes**

### 3. Enable GitHub Pages
- **Settings → Pages → Source → Deploy from branch → main → / (root) → Save**

### 4. Access Your App
```
https://YOUR-USERNAME.github.io/t2t/
```

✅ **Live in ~60 seconds. No API key needed. Completely free.**

## 🤖 How the Free AI Works

| Step | What happens |
|------|-------------|
| You upload an image | File stays in your browser — never uploaded to any server |
| TF.js loads MobileNetV2 | ~16MB model downloads once from Google CDN, then cached |
| Neural network runs | 1000 ImageNet classes classified directly in your browser |
| Keyword mapping | Class labels mapped to 12 waste material types |
| Result shown | Waste type + confidence + points + recycling tip |

**Zero API key. Zero cost. Zero data leaving your device.**
